### Lume

画像のパスが壊れる問題を調査していた。

始めはバグかと思ったけど、Lumeがデフォルトでハイフン区切りのパスをyymmddにパースする挙動だと判明した。

原因は判ったのでurl関数を書いて解決した。

```js
// _data.js
export function url(page) {
  return page.src.entry.path;
}
```
